# Idyllium для VS Code

Расширение добавляет поддержку учебного языка программирования Idyllium в VS Code.

Возможности:

- регистрация файлов `.idyl`;
- подсветка синтаксиса Idyllium;
- темы `Idyllium Dark` и `Idyllium Light` с палитрой, близкой к Web IDE;
- диагностика проекта через `IdylliumProject`;
- автодополнение модулей, методов стандартной библиотеки, пользовательских модулей и членов объектов;
- символы документа для классов, функций, полей, методов, конструкторов, переменных и `main`;
- запуск активного `.idyl` файла из VS Code;
- запуск GUI/Canvas-приложений через локальное окно предпросмотра;
- CodeLens `Run Idyllium` над `main()`;
- кнопки запуска CLI, запуска GUI и остановки в шапке редактора `.idyl`.

## Локальная сборка

Из корня репозитория:

```bash
npm run build:vscode
npm run package:vscode
```

Готовый VSIX создаётся в `packages/vscode-idyllium/idyllium-vsc-1.1.2.vsix`.
