# 🌿 Idyllium

> **Гармония синтаксиса. Простота понимания. Надёжность обучения.**

**Idyllium** — это учебный язык программирования, созданный для того, чтобы учить думать, а не зубрить исключения.

Язык спроектирован с нуля для преподавания IT-дисциплин детям школьного возраста (и не только). Idyllium избегает синтаксических ловушек, нелогичных исключений и «магии», которая заставляет новичков погружаться на уровни, к которым они пока не готовы.

> 🔗 **Попробовать прямо сейчас:** [Idyllium WebIDE](https://aumphaadr.github.io/Idyllium/)

---

## 📌 Философия

- **Приоритет — лёгкость выражения мыслей**  
  Пиши так, как думаешь. Язык не должен заставлять гадать.

- **Язык учит думать, а не зубрить исключения**  
  Каждое правило работает везде одинаково. Никаких «а вот здесь по-другому».

- **Логика важнее краткости, но код не должен «шуметь»**  
  Минимум boilerplate, максимум ясности.

- **Новичку должно быть проще понять, чем запомнить**  
  Синтаксис следует логике, а не историческим причинам.

- **Учебный язык**  
  Idyllium не претендует на звание «продакшн-инструмента», но обеспечивает плавный переход на C++, C#, Java, JavaScript и Python.

## ❌ Чего НЕТ в Idyllium

| Проблема | В других языках | В Idyllium |
|----------|----------------|------------|
| Неявное приведение типов | `7 / 4 = 1` (C++) | `7 / 4 = 1.75` (float) |
| Магия с массивами | `[1,2,3] + 4 = "1,2,34"` (JS) | Ошибка компиляции |
| Нелогичные имена функций | `randint()` / `uniform()` (Python) | `create_int()` / `create_float()` |
| Сложный рандом | `Math.floor(Math.random() * 30)` (JS) | `random.create_int(0, 29)` |
| Эльфийские письмена | `strcmp()`, `scanf("%d", &a)` | Понятные имена и ввод/вывод |
| Неопределённое поведение | `arr[5]` при размере 3 (C++) | Чёткая ошибка с пояснением |
| Классы для "Hello, World" | `public static void main(String[] args)` | `main()` и всё |

## ✅ Что Idyllium даёт взамен

### 1. Прозрачный ввод данных
```cpp
int a = console.get_int();
float b = console.get_float();
string s = console.get_string();
```
Всё логично: `get_int()` → получаем `int`, `get_string()` → получаем `string`.

### 2. Честная арифметика
```cpp
int a = 10;
int b = 3;
float c = a / b;  // 3.33333, не 3!
```
Деление всегда возвращает `float`. Для целочисленного деления есть `div()`.

### 3. Понятная работа с массивами
```cpp
array<int, 3> arr = [1, 2, 3];
int x = arr[5];  // runtime error: array index 5 out of bounds (size 3, valid indices 0-2)
```

### 4. Простые случайные числа
```cpp
int r = random.create_int(1, 6);           // 1–6
float f = random.create_float(0.0, 1.0);   // 0.0–1.0
```

### 5. Человечные сообщения об ошибках
```
main.idyl:4: error: cannot assign 'string' value to 'int' variable
```
Копируй в переводчик — и всё понятно даже без английского.

### 6. Библиотека `types` — понимание машинной арифметики
Idyllium показывает, как числа работают «под капотом» — с переполнением, двоичным представлением и фиксированной разрядностью. Это не для повседневного использования, а для понимания того, с чем вы столкнётесь в C++ и Java.

```cpp
use types;

main() {
    types.uint8 N = 252;
    N = N + 5;              // 252 + 5 = 257 → переполнение, результат 1
    string bin = N.to_bin(); // двоичное представление "00000001"
}
```

### 7. Библиотека `encoding` — мост между текстом и числами
Кодировки — вечная боль новичков. Idyllium делает работу с ними прозрачной: превращайте символы в числа и обратно, кодируйте и декодируйте строки. Никакой магии — только логика.

```cpp
use encoding;

main() {
    char ch = 'Я';
    int code = encoding.char_to_int(ch, "utf-8");  // 1071
    char restored = encoding.int_to_char(code, "utf-8");  // 'Я'
}
```

## 🧩 Что умеет Idyllium

- Переменные и типы данных (`int`, `float`, `string`, `char`, `bool`, `void`)
- Условный оператор `if` / `else` / `else if`
- Циклы `while`, `do-while`, `for`
- Статические и динамические массивы (`array`, `dyn_array`)
- Функции с параметрами по умолчанию и именованными аргументами
- Классы с наследованием, инкапсуляцией (`public`/`private`), конструкторами и деструкторами
- Библиотеки: `console`, `math`, `random`, `time`, `file`, `types`, `encoding`, `gui`
- GUI-виджеты для создания оконных приложений прямо в браузере

## 🚀 Быстрый старт

Ничего устанавливать не нужно. Всё работает в браузере.

1. Открой [Idyllium WebIDE](https://aumphaadr.github.io/Idyllium/)
2. Напиши свой первый код:
```cpp
use console;

main() {
    console.writeln("Hello, World!");
}
```
3. Нажми **Запустить**
