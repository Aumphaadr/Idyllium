GUI Урок 1: Создание интерфейса

Как собрать окно с кнопками, надписями и другими элементами управления

Подключение библиотеки gui

Для создания графического интерфейса нужно подключить библиотеку gui:

use gui;

Создание окна

Любое GUI-приложение начинается с окна:

gui.window win;
win.title = "Моё первое окно";
win.size = [300, 200]; // ширина, высота

Свойства окна:

  • title — заголовок окна;
  • size — размер в пикселях: [ширина, высота].

Виджеты: элементы интерфейса

Idyllium предоставляет следующие базовые виджеты:

  • gui.label — надпись (только для чтения);
  • gui.button — кнопка;
  • gui.text_box — поле ввода текста;
  • gui.spin_box — числовое поле со стрелками;
  • gui.check_box — флажок (галочка);
  • gui.progress_bar — индикатор прогресса.

Настройка виджетов

Каждый виджет имеет общие свойства:

  • position — позиция в окне: [x, y] (в пикселях от левого верхнего угла);
  • size — размер: [ширина, высота].

И специфические:

// Надпись
gui.label lbl;
lbl.text = "Привет!";
lbl.position = [20, 20];
lbl.size = [100, 30];

// Кнопка
gui.button btn;
btn.text = "Нажми меня";
btn.position = [20, 60];
btn.size = [120, 40];

// Числовое поле
gui.spin_box spin;
spin.min = 0;
spin.max = 100;
spin.value = 50;
spin.position = [20, 110];
spin.size = [80, 30];

// Флажок
gui.check_box chk;
chk.text = "Включить";
chk.checked = true;
chk.position = [20, 150];
chk.size = [100, 30];

// Индикатор прогресса
gui.progress_bar pb;
pb.min = 0;
pb.max = 100;
pb.value = 75;
pb.position = [20, 190];
pb.size = [200, 20];

Добавление виджетов в окно

Чтобы виджет появился в окне, его нужно добавить с помощью метода add_child():

win.add_child(lbl);
win.add_child(btn);
win.add_child(spin);
win.add_child(chk);
win.add_child(pb);

Если не вызвать add_child() — виджет не отобразится, даже если он создан и настроен.

Показ окна

После настройки всех элементов нужно:

  1. Показать окно: win.show();
  2. Запустить цикл обработки событий: gui.run();
win.show();
gui.run(); // блокирующий вызов

Без gui.run() окно мгновенно закроется.
