Урок 0: Установка Idyllium

Как подготовить среду для написания и запуска программ на Idyllium

Шаг 1: Скачайте расширение

Перейдите на страницу релизов Idyllium и скачайте последнюю версию расширения в формате .vsix:

Скачать с GitHub Releases

Файл будет называться примерно так: Idyllium-X.X.X.vsix.

Шаг 2: Откройте Visual Studio Code

Запустите редактор Visual Studio Code. Если у вас его ещё нет — установите с официального сайта.

Шаг 3: Откройте меню расширений

В левом боковом меню нажмите иконку «Extensions» (русская локализация: «Расширения»).
Либо используйте горячие клавиши: Ctrl + Shift + X (на Windows/Linux) или Cmd + Shift + X (на macOS).

Шаг 4: Установите из файла VSIX

В левом верхнем углу панели расширений найдите кнопку с тремя точками ⋯. При наведении на неё появится подсказка: «Views and More Actions».

Нажмите на неё и в выпадающем меню выберите пункт:

«Install from VSIX…» («Установить из VSIX…»)

Шаг 5: Выберите скачанный файл

В открывшемся окне проводника найдите и выберите скачанный файл Idyllium.vsix.

Visual Studio Code автоматически установит расширение.

Шаг 6: Перезапустите редактор (если потребуется)

В некоторых случаях VS Code предложит перезапустить редактор для завершения установки. Сделайте это.

После перезапуска расширение Idyllium будет полностью готово к работе.

Шаг 7: Проверьте работоспособность

  1. Создайте новую папку на компьютере (например, idyllium-test).
  2. Откройте эту папку в Visual Studio Code (File → Open Folder).
  3. Создайте в ней новый файл с расширением .idyl, например: test.idyl.
  4. Введите в него следующий код:
use console;

main() {
    console.write("Hello, World!");
}
  1. Нажмите Run Idyllium File.
  2. Если всё сделано правильно — внизу появится терминал с надписью:
Hello, World!

Поздравляем! Вы успешно установили и проверили Idyllium.
