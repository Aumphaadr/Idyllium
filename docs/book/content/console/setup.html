Самый быстрый способ: открыть IDE

Если вы только начинаете знакомиться с Idyllium, устанавливать ничего не нужно. Достаточно открыть Web IDE: там уже есть редактор, запуск программы, вывод, файловое дерево и подсветка синтаксиса.

Открыть IDE

Web IDE удобна для уроков, быстрых экспериментов и демонстраций на занятии. Открыл страницу, написал код, нажал зелёный треугольник — программа работает.

Учебник, документация и IDE живут рядом

Учебник, документация и Web IDE находятся на одном GitHub Pages-сайте. Поэтому из учебника можно открыть IDE, а из IDE - вернуться к урокам. Не нужно держать в голове два разных адреса и объяснять ученику, почему «тут теория, а практику ищи где-то ещё».

В шапке учебника есть кнопки «Открыть IDE» и «Документация». В шапке IDE есть отдельные кнопки «Учебник» и «Документация». Все три части проекта остаются рядом, но решают разные задачи.

Если хочется работать в VS Code

Для более серьёзной работы можно поставить расширение Idyllium для VS Code. Оно добавляет подсветку, запуск текущего файла и постепенно будет получать больше привычных возможностей редактора.

  1. Откройте страницу релизов: github.com/Aumphaadr/Idyllium/releases.
  2. Скачайте файл расширения с расширением .vsix.
  3. Откройте VS Code.
  4. Откройте панель Extensions.
  5. В меню этой панели выберите Install from VSIX....
  6. Укажите скачанный файл .vsix.

После установки VS Code начнёт понимать файлы .idyl. Это уже «локальный» способ работы: проекты лежат на компьютере, а не только в браузере.

Что выбрать?

  • Web IDE — лучший старт для уроков, быстрых задач и работы без установки.
  • VS Code + расширение — хороший вариант для проектов побольше и привычной файловой структуры.

Для первых уроков выбирайте Web IDE. Когда появится желание хранить проект на компьютере и работать как в обычной среде разработки, переходите в VS Code.
