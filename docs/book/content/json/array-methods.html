Что умеет json.Array

JSON-массив похож на динамический массив, но каждый его элемент является json.Value. Поэтому числа, строки и объекты сначала упаковываются в json.Value(...).

  • length() — узнать количество элементов;
  • at(index) — прочитать элемент;
  • add(value) — добавить в конец;
  • set(index, value) — заменить элемент;
  • insert(index, value) — вставить перед индексом;
  • pop() — удалить и вернуть последний элемент;
  • remove(index) — удалить элемент по индексу;
  • clear() — очистить массив.

Индексы проверяются
