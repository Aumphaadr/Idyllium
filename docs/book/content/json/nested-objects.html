Объект внутри объекта

Иногда значение слишком сложное для одной строки или одного числа. Например, позиция игрока состоит из двух координат: x и y. В JSON такое удобно хранить вложенным объектом.

Создаём вложенный объект

Вложенный объект сначала создаётся отдельно, а потом добавляется как значение в главный объект.

Читаем вложенный объект

Метод get() вернёт json.Value. Чтобы работать с ним как с объектом, используем to_object().
