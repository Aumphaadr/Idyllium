JSON-массив

JSON-массив — это список значений. В тексте JSON массив записывается в квадратных скобках, а элементы разделяются запятыми.

В обычном тексте это читается так: первый предмет — меч, второй — щит, третий — зелье. У каждого предмета есть номер позиции, но сами позиции в JSON не подписываются: порядок элементов уже хранит эту информацию.

В этом уроке будем использовать простые массивы с однотипными значениями. Так легче понять сам инструмент, не путаясь в слишком хитрых структурах.

Создадим массив предметов

В Idyllium JSON-массив создаётся типом json.Array. Добавляем элементы методом add().

["sword","shield","potion"]

Массив как значение объекта

JSON-массив может быть значением внутри объекта наравне со строкой или числом. Например, у игрока может быть имя, уровень и инвентарь.

{ "name": "Liam", "level": 7, "inventory": [ "sword", "potion" ] }

Чтение элементов массива

Элемент массива читается методом at(index). Индексы начинаются с нуля, как в обычных массивах Idyllium.

Первый предмет: sword Предметов всего: 3
