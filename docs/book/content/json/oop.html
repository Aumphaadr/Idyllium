Класс и сохранение

Если у вас уже есть класс, JSON можно использовать как мост между объектом программы и текстовым файлом сохранения.

Сделаем класс Player и добавим ему метод to_json(), который собирает свойства игрока в json.Object.

Восстановление из JSON

Обратный метод можно назвать from_json(). Он получает json.Object и раскладывает значения обратно по полям объекта.

Полный пример

{"name":"Liam","level":7} Liam: 7
