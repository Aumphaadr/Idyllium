Текст на Canvas

Для текста используется drawable.Text. Ему можно задать позицию, размер шрифта, цвет и сам текст. Без дополнительных настроек используется встроенный Source Code Pro. Если нужен конкретный шрифт, его можно загрузить через общий ресурс fonts.Font. Тот же объект подходит и для GUI-виджетов.

font выбирает рисунок букв, а font_size — их размер. Поэтому один объект main_font можно использовать у нескольких текстов разных размеров.

Файл шрифта должен лежать в проекте рядом с программой. Если файла нет, Idyllium честно скажет, что не смог его найти. Если файл существует, но не является TTF, OTF, WOFF или WOFF2, ошибка тоже появится сразу.
