События клавиатуры

Canvas умеет реагировать на клавиши. Для нажатия есть on_key_pressed, для отпускания — on_key_released.

В примере прямоугольник стоит на месте. Мы меняем только цвет его рамки: R делает рамку красной, G — зелёной, B — синей. Когда клавишу отпускают, рамка снова становится прозрачной.
