Почему первый пример нельзя было запустить дважды?

Команда CREATE TABLE players требует создать новую таблицу. При втором запуске таблица уже существует, поэтому SQLite сообщает об ошибке. Теперь мы готовы добавить условие прямо в SQL.

Конструкция IF NOT EXISTS читается как «если не существует». При первом запуске таблица будет создана, а при следующих команда спокойно оставит существующую таблицу на месте. Данные внутри неё не удалятся.

Проверка перед удалением

Обратная конструкция IF EXISTS пригодится при удалении таблицы. Эту команду удобно использовать в коротких учебных экспериментах, но в настоящем проекте она безвозвратно удалит и таблицу, и все её строки.

Теперь понятна строка, которую мы раньше иногда встречали в готовых примерах: она очищала место перед повторным созданием учебной таблицы.
