Ваш первый код на Idyllium

Для вывода классического "Hello, World!" наберите следующий код:

// hello.idyl
use console;

main() {
    console.write("Hello, World!", '\n');
    console.write("Это ваша первая программа на Idyllium!", '\n');
}

Разберём ключевые моменты:

  • Функция console.write() используется для вывода данных в консоль.
  • Она принимает неограниченное количество аргументов, разделённых запятыми. Текст, числа, переменные — всё можно передать в один вызов.
  • Каждая команда в Idyllium, как и во многих C-подобных языках, завершается точкой с запятой (;). Это помогает компилятору точно понимать, где заканчивается инструкция.

Почему в конце стоит '\n'?

В отличие от print() в Python, функция console.write() не делает автоматический перенос строки. Это даёт вам полный контроль над форматом вывода.

Рассмотрим пример без переноса:

// hello.idyl
use console;

main() {
    console.write("Hello");
    console.write("World");
}

Этот код выведет всё в одну строку:

HelloWorld

Чтобы получить перенос, нужно явно указать управляющую последовательность '\n'. Её можно добавить как отдельный аргумент или приклеить к строке — оба способа работают одинаково.

Гибкое использование '\n'

Символ перехода на новую строку можно размещать в любом месте внутри console.write():

// hello.idyl
use console;

main() {
    console.write("Hello", '\n', "my", '\n', "Lord");
}

Результат выполнения:

Hello my Lord

Как видите, '\n' работает именно там, где вы его разместили — не раньше и не позже.

Вывод чисел

Idyllium умеет выводить не только текст, но и числа — без дополнительных преобразований:

// hello.idyl
use console;

main() {
    console.write("Сейчас выведем несколько чисел:\n");
    console.write(11, '\n', 22, '\n', 33, '\n', 44);
}

Результат:

Сейчас выведем несколько чисел: 11 22 33 44

Обратите внимание: текстовые данные всегда заключаются в кавычки ("..."), а числовые значения пишутся без них. Это помогает Idyllium точно отличать строку от числа — и не делать догадок за вас.

Теперь вы готовы не просто писать код, а управлять тем, что видит пользователь. Добро пожаловать в Idyllium!
