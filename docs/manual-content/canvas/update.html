Один раз и каждый кадр

У Canvas есть две особенно важные функции-события. on_init вызывается один раз при запуске, а on_update вызывается снова и снова, пока приложение работает.

Это похоже на театр: в on_init мы расставляем декорации, а в on_update двигаем актёров по сцене.

Свойство framerate_limit ограничивает количество обновлений в секунду. Если поставить 60, Canvas будет стараться обновляться не чаще шестидесяти раз в секунду.
