Когда кода становится много

Canvas-программа быстро разрастается: отдельно настройка объектов, отдельно клавиши, отдельно мышь, отдельно обновление кадра. Чтобы не заблудиться, часть кода можно вынести в другой файл.

main.idyl

scene.idyl

Новых возможностей здесь нет. Мы просто разложили код по файлам: главный файл собирает окно, а scene.idyl отвечает за поведение Canvas.
