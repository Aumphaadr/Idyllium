Таблица есть. Где данные?

В прошлом уроке мы создали таблицу players, но пока она пуста. Для изменения данных в SQL чаще всего используются три команды: INSERT добавляет строки, UPDATE изменяет их, а DELETE удаляет.

Добавляем игроков

Сначала перечисляются колонки, затем — значения в том же порядке. Текст в SQL записывается в одинарных кавычках, а числа — без кавычек.

В этих командах значения придуманы автором программы и написаны прямо в коде. Пользовательский ввод так подставлять нельзя — безопасный способ мы изучим через два урока.

Условие WHERE выбирает строки с именем Liam. Без него команда изменила бы уровень всех игроков. Метод execute() возвращает sqlite.Result, а свойство affected_rows сообщает, сколько строк действительно изменилось.

Удаляем запись

Здесь снова важно условие WHERE. Команда DELETE FROM players без условия удалит из таблицы вообще всех игроков. SQL послушен буквально: он не угадывает, что программист «наверное имел в виду».
