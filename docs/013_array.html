Урок 13: Массивы фиксированной длины

Как хранить много однотипных данных без сотни переменных

Проблема: много однотипных данных

Представим, что у вас есть семь чисел, и их нужно вывести:

int a = 29;
int b = 84;
int c = 16;
int d = 35;
int e = 43;
int f = 60;
int g = 17;

console.write(a, '\n');
console.write(b, '\n');
console.write(c, '\n');
console.write(d, '\n');
console.write(e, '\n');
console.write(f, '\n');
console.write(g, '\n');

Этот код громоздкий, неудобный и не масштабируется. Если чисел будет 100 — никто не будет писать 100 переменных.

Что такое массив?

Массив — это набор ячеек памяти компьютера, предназначенных для хранения данных.

Напомним: переменная — это ячейка памяти компьютера, предназначенная для хранения данных.

Разница в масштабе: переменная хранит одно значение, а массив — много значений (по одному в каждой ячейке).

Создание массива

В Idyllium массив объявляется следующим образом:

array<int, 7> nums = [29, 84, 16, 35, 43, 60, 17];

Разберём:

  • array — ключевое слово, обозначающее «статический массив»;
  • <int, 7> — в угловых скобках указывается тип данных в ячейках (int) и количество ячеек (7);
  • nums — имя массива;
  • [29, 84, ..., 17] — начальные значения (можно не указывать — тогда все ячейки будут заполнены дефолтными значениями).

Длина массива фиксирована и не может быть изменена после объявления.

Сходство с переменной

Синтаксис создания массива напоминает создание переменной:

// Переменная:
int N = 65;

// Массив на 7 ячеек:
array<int, 7> nums = [29, 84, 16, 35, 43, 60, 17];

В обоих случаях есть:

  • Тип — int vs array<int, 7>;
  • Имя — N vs nums;
  • Значение — 65 vs [29, 84, 16, 35, 43, 60, 17].

Массив — это расширенная форма переменной.

Сравнение с переменной: создание, редактирование, вывод

int A; // создание переменной, значение = 0
array<int, 50> B; // создание массива из 50 нулей

int N = 38; // создание с указанием значения
array<int, 4> P = [29, 84, 16, 35];

N = 51; // редактирование переменной
P[0] = 72; // редактирование ячейки массива

console.write(N, '\n'); // вывод переменной
console.write(P, '\n'); // вывод всего массива
console.write(P[2], '\n'); // вывод второй ячейки
51 [29, 84, 16, 35] 16

Нумерация ячеек

Ячейки массива нумеруются с нуля, как во всех современных языках программирования:

array<int, 6> K = [81, 45, 33, 27, 90, 64];

console.write(K[0]); // 81
console.write(K[1]); // 45
console.write(K[2]); // 33
console.write(K[3]); // 27
console.write(K[4]); // 90
console.write(K[5]); // 64
console.write(K[6]); // ОШИБКА: нет шестой ячейки

Если в массиве 6 ячеек — они пронумерованы от 0 до 5. Попытка обратиться к ячейке 6 вызовет ошибку выполнения.

Преимущество: перебор циклом

С переменными пришлось бы писать:

int a = 29;
int b = 84;
int c = 16;
int d = 35;
int e = 43;
int f = 60;
int g = 17;

console.write(a, '\n');
console.write(b, '\n');
console.write(c, '\n');
console.write(d, '\n');
console.write(e, '\n');
console.write(f, '\n');
console.write(g, '\n');

С массивом — гораздо короче и легко масштабируется:

array<int, 7> nums = [29, 84, 16, 35, 43, 60, 17];
for (int i = 0; i < 7; i = i + 1) {
    console.write(nums[i], '\n');
}

Если чисел будет 1000 — достаточно изменить лишь число в условии цикла.

Массивы других типов

Массивы не ограничены типом int. Можно создавать массивы любых типов:

array<float, 5> fnums = [3.6, 2.14, 7.0, -5.91, 8.33];
array<string, 3> tools = ["Кирка", "Топор", "Удочка"];
array<bool, 4> check = [true, true, false, true];

Итог: что нужно знать о массивах

// Создание с указанными значениями
array<int, 5> P = [24, 91, 63, 50, 78];

// Создание с дефолтными значениями (0.0 для float)
array<float, 8> K;

// Редактирование ячейки
P[0] = 25;

// Чтение значения из ячейки
int A = P[1];

// Заполнение циклом
for (int i = 0; i < 8; i = i + 1) {
    K[i] = random.create_float(-3, 3);
}

// Вывод циклом
for (int i = 0; i < 5; i = i + 1) {
    console.write(P[i]);
}

Бонус: типичные ошибки

Операции, которые работают с переменной, не работают со всем массивом сразу:

array<int, 5> P = [78, 91, 63, 50, 24];
if (P > 50) { // ОШИБКА: нельзя сравнивать массив с числом }
if (P[0] > 50) { // OK: сравниваем ячейку (78 > 50) }

console.write(P + 4); // ОШИБКА: нельзя складывать массив и число
console.write(P[1] + 4); // OK: выведется 95

Массив — это контейнер. Работать с данными нужно поэлементно — через перебор ячеек.
