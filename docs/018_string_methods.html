Урок 18: Методы строк

Анализ, изменение и разбор текста — без регулярных выражений и путаницы

Методы, возвращающие bool

Эти методы отвечают на вопрос «да или нет».

.contains(c)

Проверяет, есть ли подстрока или символ в строке:

string s = "Hello, World!";
bool has_o = s.contains('o');      // true
bool has_abc = s.contains("abc");   // false

.is_int() и .is_float()

Проверяют, можно ли строку безопасно преобразовать в число:

string a = "123";
string b = "12.5";
string c = "abc";

console.write(a.is_int(), '\n');     // true
console.write(b.is_float(), '\n');  // true
console.write(c.is_int(), '\n');     // false

Методы, возвращающие int

Эти методы работают с позициями и количеством.

.find(c)

Возвращает индекс первого вхождения символа или подстроки, или -1, если не найдено:

string t = "Кот и собака";
int idx1 = t.find('о');      // 1
int idx2 = t.find("собака"); // 6
int idx3 = t.find("рыба");   // -1

.count(c)

Считает, сколько раз встречается символ или подстрока:

string u = "абракадабра";
int a_count = u.count('а'); // 5
int abra_count = u.count("абра"); // 2

Методы, возвращающие string

Эти методы создают новую строку на основе исходной.

.toUpperCase() и .toLowerCase()

string v = "Привет, Мир!";
console.write(v.toUpperCase(), '\n'); // ПРИВЕТ, МИР!
console.write(v.toLowerCase(), '\n'); // привет, мир!

.substring(start, end)

Возвращает часть строки от индекса start до end включительно:

string w = "Программирование";
console.write(w.substring(0, 3), '\n'); // "Про"
console.write(w.substring(5), '\n');      // "аммирование" (до конца)
console.write(w.substring(-1, 3), '\n'); // "Про" (отрицательный start → 0)

.replace(old, new)

Заменяет все вхождения подстроки old на new:

string x = "Кот и кот и ещё кот";
console.write(x.replace("кот", "пёс"), '\n');
Кот и пёс и ещё пёс

Обратите внимание: первый "Кот" не был заменён из-за того, что функция является регистрозависимой.

Метод, возвращающий массив

.split(c)

Разбивает строку по разделителю и возвращает динамический массив строк:

string data = "яблоко,банан,апельсин";
dyn_array<string> fruits = data.split(",");
console.write(fruits, '\n');
["яблоко", "банан", "апельсин"]

Метод особенно полезен при обработке CSV-подобных данных.
