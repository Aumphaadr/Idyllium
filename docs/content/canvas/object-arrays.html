Массивы объектов

В массиве можно хранить не только числа или строки, но и drawable-объекты. Это удобно, когда на экране много похожих объектов.

Здесь пять прямоугольников стоят столбиком. При нажатии R каждый из них поворачивается на несколько градусов.

Метод rotate() поворачивает объект ещё на указанное количество градусов. Если вызвать rotate(8) несколько раз, угол будет накапливаться.
