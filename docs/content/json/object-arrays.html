Вернёмся к нескольким игрокам

В первом уроке мы хранили имена и уровни в разных файлах и разных массивах. JSON позволяет хранить каждого игрока как отдельный объект, а всех игроков — как массив объектов.

Создаём массив игроков

Сначала создаём объекты игроков, затем добавляем их в общий JSON-массив.

Читаем массив игроков

Каждый элемент массива — это json.Value. Мы превращаем его в json.Object и достаём нужные поля.
