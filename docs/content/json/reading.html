Прочитаем JSON из файла

Для чтения JSON удобно забрать весь файл одной строкой через read_all(), а затем передать текст в json.parse().

Пусть в файле player.json лежит такой JSON:

{"name":"Liam","age":12,"is_admin":false,"middle_name":null}

Достаём отдельные свойства

Метод get() достаёт значение по ключу. Но возвращает он не сразу строку или число, а json.Value. После этого мы явно говорим, какой тип ожидаем.

Liam: 12, admin = false

Если тип заранее неизвестен

Иногда программа не уверена, что именно лежит в JSON. Тогда можно сначала проверить тип, а уже потом преобразовывать значение.

Целое число: 12

Что такое null

null означает «значения нет». Это не пустая строка, не ноль и не false. Это отдельное состояние: ячейка есть, но внутри неё ничего не лежит.

Значения нет

null нельзя превратить в строку, число или булево значение. Если попробовать, программа выдаст понятную ошибку времени выполнения.
