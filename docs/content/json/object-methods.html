Что умеет json.Object

JSON-объект хранит пары «ключ — значение». Ключи являются строками, а значения являются json.Value.

  • length() — количество ключей;
  • has(key) — проверить наличие ключа;
  • get(key) — получить значение;
  • add(key, value) — добавить новый ключ;
  • set(key, value) — изменить существующий ключ;
  • remove(key) — удалить ключ;
  • keys() — получить динамический массив ключей.

Смотрим ключи

["name", "level", "money"] Всего ключей: 3

Проверка перед чтением

Если ключ может отсутствовать, сначала используйте has(). Так программа сможет принять решение сама, а не упасть с ошибкой.
