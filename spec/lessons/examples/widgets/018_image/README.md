# Урок 18: Картинки в GUI

Draft-урок для новой спецификации IdylliumNext.

Он фиксирует виджет `gui.Image`, который показывает картинку внутри GUI-окна.

Основной стиль:

- создать `gui.Image`;
- задать позицию и размер через `x`, `y`, `width`, `height`;
- загрузить файл через `load_from_file`;
- при необходимости выбрать режим масштабирования через `resize_mode`;
- добавить картинку в окно через `win.add_child(image)`.

Поддерживаемые значения `resize_mode`:

- `"fit"` - картинка помещается целиком и сохраняет пропорции;
- `"fill"` - картинка заполняет весь виджет и сохраняет пропорции, лишнее обрезается;
- `"stretch"` - картинка растягивается точно под размер виджета, пропорции могут исказиться;
- `"original"` - картинка рисуется в исходном размере от левого верхнего угла виджета.

Если файл не найден, `load_from_file` должен выдавать читаемую runtime error.
