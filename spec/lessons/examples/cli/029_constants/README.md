# Урок 29: именованные константы

Ручное дополнение к frozen lesson spec. Оно фиксирует утверждённую семантику
`const`; соответствующие позитивные и негативные проверки находятся в
`tests/smoke.test.ts`.

- локальные и верхнеуровневые константы требуют инициализатор;
- прямое и составное переприсваивание запрещено;
- верхнеуровневая константа пользовательского модуля доступна через
  `module.NAME`;
- для массивов и объектов `const` защищает привязку имени, но не замораживает
  содержимое рекурсивно.
